# Iconos de la Extensión

Esta carpeta debe contener el icono de la extensión.

## Requisitos

- **Nombre del archivo:** `icon.png`
- **Tamaño recomendado:** 128x128 píxeles (mínimo)
- **Formato:** PNG con fondo transparente
- **Tamaño máximo:** 256x256 píxeles

## Cómo crear el icono

1. Crea una imagen de 128x128 o 256x256 píxeles
2. Usa un fondo transparente
3. Guárdala como `icon.png` en esta carpeta

## Herramientas recomendadas

- [Figma](https://figma.com) (gratis)
- [Canva](https://canva.com) (gratis)
- [GIMP](https://gimp.org) (gratis)
- Adobe Illustrator / Photoshop

## Ejemplo de estructura

```
images/
└── icon.png    ← Tu icono aquí
```

---

**Nota:** Sin el icono, la extensión se publicará con un icono genérico.
